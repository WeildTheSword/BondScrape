When you drop a NOS PDF, the system extracts structured data from the raw text. Choose how that extraction happens:

Local Mode
No API Key
Heuristic extraction using regex pattern matching on the raw PDF text. Captures core fields (issuer, par amount, dates, maturity schedule, bond type, basis of award, call provisions). Complex fields like coupon constraints or issue price requirements may be incomplete. Best for demos and testing.
LLM Mode
API Key Required
Full extracted text is sent to Claude (Anthropic API) with the 55-feature JSON schema. Returns complete structured extraction with high accuracy across all fields. Requires an Anthropic API key.
Your key is stored in your browser only (localStorage). Never sent anywhere except the Anthropic API.

Pre-loaded test documents from the sidebar always use their validated ground truth data regardless of mode. This setting only affects newly dropped PDFs.
